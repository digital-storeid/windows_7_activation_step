Cara Aktivasi Windows 7

1.       Tekan tombol Windows pada keyboard , kemudian ketik Activate Windows

 


2.       Muncul tampilan wizard berikut, silahkan masukan Product Key yang dikirim.


3.       Setelah masukan key klik Next, kemudian tunggu hingga proses aktivasi selesai.


 

4.       Proses aktivasi sedang berlangsung tunggu hingga selesai. Wajib terhubung dengan internet.

 


5.       Aktivasi sukses dan permanen aktif secara lifetime, Klik Close. SELESAI.


 

NOTE :

Product key dapat digunakan untuk keperluan install ulang. Kecuali hardwarenya ganti atau upgrade seperti Hardisk, SSD, RAM, Fan, dll.

Maka product key tidak dapat digunakan lagi dan key hangus.

 

+===========================================================================================================+

JIKA SAAT AKTIVASI MUNCUL PESAN ERROR BISA IKUT INSTRUKSI BERIKUT.

+===========================================================================================================+

1.       Pesan error saat aktivasi itu wajar karena ini windows 7 bukan windows 10 atau 11.

Klik Show me other ways to activate.


 

2.       Klik Use the automated phone system.


 

3.       Ada tampilan negara pilih Afghanistan, lalu klik Next.


 

 

4.       Silahkan Foto atau Screenshoot, atau bisa juga ketikan kode dari kolom 1 9.

Contoh : 008071-094300-777135-459564-384163-626196-035865-258273-640102


 

Kemudian tolong kirim ke chat shopee atau kalau gagal terkirim bisa balas ke email ini.

 

5.       Tunggu beberapa menit kami akan segera kirimkan kode Konfirmasi ID untuk proses aktivasi Windowsnya.

6.       Setelah menerima kode Konfirmasi ID dari kami silahkan dimasukan kedalam kolom A-H. Setelah dimasukan kodenya klik Next.

 

Contoh : 450802-283070-172591-019212-980843-052374-170574-426595


 

7.       Aktivasi sukses dan permanen aktif secara lifetime. Klik Close. SELESAI.
